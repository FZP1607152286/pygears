Filter¶

filt(*args, **kwds)¶

Filters the data field of Union type, by passing it forward only if it carries the data Union subtype whose index is equal to the value supplied to the sel field. Index of a Union subtype is equal to the position of the subtype within the Union definition.

Example - Lets make a Union of three types:

data_t = Union[Uint[1], Uint[2], Uint[3]]

We are going to send three values to the filt gear via data field, each of them with a different type:

data = [
    data_t(Uint[1](0), 0),
    data_t(Uint[2](3), 1),
    data_t(Uint[3](7), 2)
]

For each input data, the sel field should contain the index of the Union subtype which is to be forwarded by the filt gear. Other subtypes will be filtered (blocked). For an example, if we would like that only the data of the subtye Uint[2] is forwarded (subtype index is 1), the following values should be passed via sel field:

sel = [1, 1, 1]

We can drive these two fields as a single Tuple in the following way:

drv(t=Tuple[data_t, Uint[2]], seq=zip(data, sel)) \
    | filt

The output sequence of the filt gear should contain a single element:

[(Uint[3](3), Uint[2](1))]
