Typing¶

class pygears.typing.Integer(val: int = 0)¶

Base type for both Int [N] and Uint [N] generic types.

class pygears.typing.Int(val: int = 0)¶

Fixed width generic signed integer data type.

Generic parameters:
N: Bit width of the Int [N] representation
Args:
val: Integer value to convert to Int [N]

Int [N] is a generic datatype derived from Integer [N]. It represents signed integers with fixed width binary representation. Concrete data type is obtained by indexing:

>>> i16 = Int[16]
class pygears.typing.Uint(val: int = 0)¶

Fixed width generic unsigned integer data type.

Generic parameters:
N: Bit width of the Uint [N] representation
Args:
val: Integer value to convert to Uint [N]

Uint [N] is a generic datatype derived from Integer [N]. It represents unsigned integers with fixed width binary representation. Concrete data type is obtained by indexing:

>>> u16 = Uint[16]
class pygears.typing.Array¶

Generic container datatype that holds N instances of type T

Generic parameters:

T: Type of the Array [T, N] elements

N: Number of elements in the Array [T, N]

Concrete data type is obtained by indexing:

>>> u16_4 = Array[Uint[16], 4]
