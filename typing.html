class pygears.typing.base.GenericMeta¶

Bases: pygears.typing.base.TypingMeta

Base class for all types that have a generic parameter.

__eq__(other)¶

Return self==value.

args¶

Returns a list of values supplied for each generic parameter.

>>> assert Tuple[Uint[1], Uint[2]].args == [Uint[1], Uint[2]]
base¶

Returns base generic class of the type.

>>> assert Uint[16].base == Uint
fields¶

Returns the names of the generic parameters.

>>> assert Tuple[Uint[1], Uint[2]].fields == ('f0', 'f1')
>>> assert Tuple[{'u1': Uint[1], 'u2': Uint[2]}].fields == ('u0', 'u1')
is_generic()¶

Return True if no generic parameter of the type was supplied a value.

>>> assert Uint.is_generic() == True
>>> assert Uint[16].is_generic() == False
is_specified()¶

Return True if all generic parameters were supplied concrete values.

>>> assert Uint['template'].is_specified() == False
>>> assert Uint[16].is_specified() == True
templates¶

Returns a list of templated generic variables within the type. The type is searched recursively. Each template is reported only once.

>>> assert Tuple[Tuple['T1', 'T2'], 'T1'].templates == ['T1', 'T2']

class pygears.typing.uint.Int(val: int = 0)¶

Bases: pygears.typing.uint.Integer

Fixed width generic signed integer data type.

Generic parameters:
N: Bit width of the Int [N] representation
Args:
val: Integer value to convert to Int [N]

Int [N] is a generic datatype derived from Integer [N]. It represents signed integers with fixed width binary representation. Concrete data type is obtained by indexing:

>>> i16 = Int[16]

class pygears.typing.array.ArrayMeta¶

Bases: pygears.typing.base.EnumerableGenericMeta

__getitem__(index)¶

If a single element is supplied for index, returns type T. If a slice is suplied for index, an Array type is returned with a number of elements equal to the slice size.

>>> assert Array[Uint[2], 5][3] == Uint[2]
>>> assert Array[Uint[2], 5][2:4] == Array[Uint[2], 2]
keys()¶

Returns a list of keys that can be used for indexing Array [T, N] type. Number of keys equals to the number of elements N.

>>> assert Array[Uint[2], 5].keys() == [0, 1, 2, 3, 4]

Generic container datatype that holds N instances of type T

Generic parameters:

>>> u16_4 = Array[Uint[16], 4]
